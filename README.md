# 365INO Enterprise Website

Production-ready enterprise website for 365 Innovations Corporation (365INO), built with Next.js App Router, React, TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Included

- Home, About, Services, Solutions, AI & Automation, Digital Transformation, Government Solutions, Commercial Solutions, Industries, Case Studies, Leadership, Careers, Contact, Privacy Policy, Terms of Use, and 404 pages
- Responsive navigation, footer, hero, CTA, statistics counters, service cards, technology grid, timeline, FAQ, contact form, and scroll progress
- Per-page metadata, OpenGraph and Twitter metadata, structured organization data, robots.txt, sitemap.xml, and favicon placeholder
- Accessibility-minded markup, keyboard focus states, high-contrast colors, and reduced-motion support

## Deployment

The site can deploy to Vercel or any platform that supports Next.js 15. Set the production domain in `lib/metadata.ts` and `app/layout.tsx` if deploying somewhere other than `https://365ino.com`.
